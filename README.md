# 📁 Celeste Bazzi - Portfolio Progetti

Welcome to my academic project repository. This collection includes software, simulations, and analyses developed primarily in the fields of Artificial Intelligence, Computer Vision, Augmented Reality, Natural Language Processing (NLP) and Symbolic and Distributed Artificial  Intelligence (S&DAI).

Each project is organized in a dedicated folder and includes the related **explanatory PDF** for a deeper understanding of the technical architecture and the results obtained.

## Project index

### 🐢 1. The Turtle Labyrinth (Augmented Reality)
An educational augmented reality game developed for Android devices. The app introduces users to basic programming concepts by controlling the movements of the turtle "Tilly" using commands inspired by the LOGO language.
* **Technologies & Tool:** Unity, Vuforia AR Engine, C#, Blender, AI Navigation, Google ARCore.

### ✍️ 2. Writer Identification (Computer Vision)
Study and implementation of computer vision algorithms for author recognition of handwritten text (comparison between free text and signatures (real and imitation)). The project evaluates and compares the effectiveness of feature extraction via SIFT and the Bag of Words (BoW) approach applied to different preprocessing techniques (Canny filter vs Variable Threshold).
* **Technologies & Tool:** Python, OpenCV, SIFT, K-Means Clustering, Cosine Similarity.

### 🃏 3. Pokémon Card Game Simulator (NLP & Multi-Agent Systems)
An intelligent system for automatically simulating the Pokémon card game. The project includes an NLP pipeline (web scraping with BeautifulSoup, POS Tagging, and lemmatization with the Stanza library) to extract and interpret the semantic texts of card effects, translating them into game rules. The logic simulation is managed via a multi-agent architecture in JADE and features a dynamic graphical interface.
* **Technologies & Tool:** Java, JADE (Java Agent Development Framework), JavaFX, Python (BeautifulSoup, Stanza NLP), Maven.
